# Git Scripts

This is a group of scripts to make the git workflows faster.

## Installation

Clone the file:

```bash
git clone git@github.com:fabian818/git-scripts.git
```

Run the install script:

```bash
sh install.sh
```

## Usage

```bash
gcommit "ADD: my awesome feature"

gpush my-awesome-branch

grebase main
```
